# Smart Task List

Aplicação de gerenciamento de tarefas desenvolvida com foco em lógica de programação e experiência do usuário.

## Funcionalidades
- Adicionar tarefas
- Remover tarefas
- Marcar como concluída
- Filtros de tarefas
- Contador de tarefas
- Tema dark/light
- Persistência com localStorage

## Tecnologias
- HTML
- CSS
- JavaScript
